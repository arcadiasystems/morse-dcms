# morse-sdk

TypeScript SDK for [Morse dCMS](../README.md) on the [Sui](https://sui.io/) blockchain.

> **Status: 0.0.0 - unreleased.**
> This package is under active development. The public API is not stable and
> there is no published release yet. Do not use in production.

## Install

Not yet published to npm.

## Development

From the repository root:

```sh
bun install
```

From `morse-sdk/`:

```sh
bun run lint        # biome check
bun run typecheck   # tsc --noEmit
bun run test        # bun test
bun run build       # tsc (emits dist/)
```
